## About

### Dependencies
```bash
brew install tmux
```

### Installing

```bash
git clone https://github.com/jairovm/dotfiles.git ~/.dotfiles
ln -nsf ~/.dotfiles/tmux/tmux.conf ~/.tmux.conf
```
